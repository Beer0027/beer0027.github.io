About Me

My name is Barret Beermann. I am a graduate of the BHTM program from Algonquin College and I am currently a 1st level IMD student. I have enrolled in this program because I am interested in pursuing coding professionally, but I am also interested in learning other aspects of media design such as digital graphics.

The reason that I became interested in coding was because of the CompSci class I took in high school. We explored c# using Visual Studio (thankfully we are using it in this class). I learned how to code little programs and mini games, but I definitely couldn't do that anymore.
